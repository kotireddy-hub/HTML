# bdo
1. it is used to display text rtl(right to left) or ltr (left to right)
2. pair tag
3. rtl to display text in right to left
4. ltr to display text in left to right

### Syntax

```
<bdo dir="rtl"> your text here </bdo>
```
#### Sample Code for bdo 
```
<bdo dir="rtl">HI</bdo>
```
